# Draw a family tree in R/kinship2

## Requirements
* R
* [kinship2 package](http://cran.r-project.org/web/packages/kinship2/index.html)

## Usage

    R -q -f familytree.R --args familytree.csv

## Output
Open familytree.pdf in your working directory.
